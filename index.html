Урок 1 - Введение. Скачиваем софт

Вопросы по уроку:

  1. Что такое верстка сайта(объясните своими словами)?
  2. Из чего состоит сайт?
  3. Что такое .psd файл?
  4. Что такое HTML теги, для чего они нужны и на какие 2 группы разделенны?
  5. Что такое атрибуты тегов и для чего они нужны?
  6. Опишите базовую разметку HTML-страницы.
  7. Для чего нужен тег <!DOCTYPE html>?
  8. Для чего нужен тег <head> и какие теги он содержит внутри себя?
  9. Для чего нужен тег <body>?

Ответы на вопросы:

  1. Верстка сайта это создание его внешнего вида, то как он будет выглядеть.
  2. Сайт состоит из нескольких состовляющих: дизайн, вёрстка сайта, программного кода, клиент-серверной части, контента сайта, хостинга, доменного имени и CMS.
  3. Это файл в котором обычно представлен макет страницы сайта или нескольких страниц сайта, который нужен для верстки.
  4. HTML теги используются для разграничения начала и конца элементов в разметке. Они бывают парные и не парные.
  5. Атрибуты тегов - это элементы которые видоизменяют содержимое тега. Они заключаются в кавычки " ".
  6. HTML страница в обычном представлении имеет header, footer, content часть, menu, side bar один или несколько.
  7. Этот тег нужен, чтобы браузер сразу понимал, какую версию html мы используем.
  8. Тег head предназначен для хранения других элементов, цель которых — помочь браузеру в работе с данными. Также внутри контейнера <head> находятся метатеги, которые используются для хранения информации предназначенной для браузеров и поисковых систем. Например, механизмы поисковых систем обращаются к метатегам для получения описания сайта, ключевых слов и других данных.
  9. В теге <body> хранится вся разметка страницы.

Ответы на вопросы:

  1. Тег <a> имеет атрибуты:
    1. Download - Предлагает скачать указанный по ссылке файл.
    2. href - Задает адрес документа, на который следует перейти.
    3. target - Имя окна или фрейма, куда браузер будет загружать документ.
    4. name - Устанавливает имя якоря внутри документа.
  2. Существует 2 вида путей к файлам - с открытием файла в этом же окне и открытие файла в новом окне
  3. Тег <img> добавляет на страницу изображение
  4. В верстке обычно используется .jpg .png форматы изображений. Изображение формата .png имеет альфа канал, который
    позволяет при верстке фону изображения принимать фон блока в котором он находится. Изображение формата .jpg имеет
    свой фон, который не зависит от фоно используемого в данном блоке.

Урок 4 - Изучаем HTML - таблицы

Вопросы по уроку:

  1. Что обозначают теги <table> <tr> и <td> ?
  2. Перечислите все атрибуты для тега <table> и их предназначения.
  3. При помощи каких атрибутов можно объеденять несколько ячеек таблицы в одну ?
  4. При помощи каких тегов мы можем объеденять блоки таблицы по смыслу ?

Ответы на вопросы:

  1. Тег <table> служит для создания таблиц в HTML верстке. Теги <tr> и <td> нужны для создания строк и столбцов таблицы
  2. Атрибуты для тега <table>:
    • align - Определяет выравнивание таблицы.
    • background - Задает фоновый рисунок в таблице.
    • bgcolor - Цвет фона таблицы.
    • border - Толщина рамки в пикселах.
    • bordercolor - Цвет рамки.
    • cellpadding - Отступ от рамки до содержимого ячейки.
    • cellspacing - Расстояние между ячейками.
    • cols - Число колонок в таблице.
    • frame - Сообщает браузеру, как отображать границы вокруг таблицы.
    • height - Высота таблицы.
    • rules - Сообщает браузеру, где отображать границы между ячейками.
    • summary - Краткое описание таблицы.
    • width - Ширина таблицы.
  3. Объеденять ячейки можно при помощи атрибута rowspan(colspan).
  4. мы можем объеденять блоки таблицы по смыслу в теги <thead>,<tbody>,<tfoot>.

Урок 5 - Изучаем HTML - form, input, textarea, button

Вопросы по уроку:

  1. Что такое HTML формы и для чего они нужны ?
  2. Что обозначает тег form ?
  3. Что такое input ?
  4. Что такое select и option?
  5. Что такое textarea ?
  6. Для чего нужны атрибуты type, value, disabled, name, multiple, placeholder, action ?
  7. Какие типы input существуют (тип указывается в атрибуте type="") ? Написать краткое пояснения по каждому.

Ответы на вопросы:

  1. Форма предназначена для обмена данными между пользователем и сервером.
  2. Тег <form> устанавливает форму на веб-странице.
  3. Тег <input> является одним из разносторонних элементов формы и позволяет создавать разные элементы интерфейса и
    обеспечить взаимодействие с пользователем. Главным образом <input> предназначен для создания текстовых полей,
    различных кнопок, переключателей и флажков.
  4. Тег <select> позволяет создать элемент интерфейса в виде раскрывающегося списка, а также список с одним или
    множественным выбором. Каждый пункт создается с помощью тега <option>, который должен быть вложен в контейнер <select>.
  5. Поле <textarea> представляет собой элемент формы для создания области, в которую можно вводить несколько строк текста.
    • type - Сообщает браузеру, к какому типу относится элемент формы.
    • value - Значение элемента.
    • disabled - Блокирует доступ и изменение элемента.
    • name - Имя поля, предназначено для того, чтобы обработчик формы мог его идентифицировать.
    • multiple - Позволяет загрузить несколько файлов одновременно.
    • placeholder - Выводит подсказывающий текст.
    • action - Адрес программы или документа, который обрабатывает данные формы.
  7. Основной атрибут тега <input>, определяющий вид элемента — type. Он позволяет задавать следующие элементы формы:
    текстовое поле (text), поле с паролем (password), переключатель (radio), флажок (checkbox), скрытое поле (hidden),
    кнопка (button), кнопка для отправки формы (submit), кнопка для очистки формы (reset), поле для отправки файла (file)
    и кнопка с изображением (image).
